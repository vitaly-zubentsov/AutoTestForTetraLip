1)	Значение Time of position для long lip type 3 в программе не меняется. Следовательно, поле для ввода данного значения отсутствует, по умолчанию отправляется сообщение с данными для 10 числа месяца, 10 часов, 10 минут, 10 секунд.
2)	Для всех типов long lip значение Location data равно 5 (Location circle with altitude)
3)	Для всех типов long lip значение Location shape равно 5 (Location circle with altitude)
4)	Программа не проверяет вводимые данные и является просто генератором сообщения, без проверки могут ли существовать данные значения в действительности.
5)	Для помощи при составлении сообщений lip необходимо изучить документ ETSI TS 100 392-18-1 стандарта ETSI TETRA.
6)	Программа позволяет генерировать только три типа сообщения long lip, более подробно про три типа смотри стандарт.
7)	Сообщения посылаются в соответствии с таймером, при этом сообщение alive отправляется раз в 25 секунд, но только при отправке очередного информационного сообщения.
